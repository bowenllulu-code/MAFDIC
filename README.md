# MAFDIC

MAFDIC is short for **Multi-Agent Fund Distribution Insight Claw**.

MAFDIC is a claw-style, multi-agent fund distribution operation system for institutional investor operations. It combines operational configuration, transaction and revenue analytics, opportunity tracking, reporting automation, and an intelligent assistant that can collaborate with users through conversation.

## Target Users

- Dedicated fund distribution operations staff
- Senior managers and business decision makers
- Data and reporting owners supporting institutional investor operations

## Core Capabilities

- Investor operation data statistics and analysis
- Investor advance funding configuration
- Advance funding bank maintenance
- Interest accrual configuration maintenance
- Investor transaction record query and analysis
- Fund holding analysis
- Revenue data analysis
- Sales opportunity management
- Performance analysis generated from opportunity-linked transactions
- Conversational intelligent assistant
- Scheduled task configuration
- Report generation and delivery
- Email and notification push

## Product Direction

MAFDIC is intended to become an operational workbench rather than a single-purpose dashboard. The system should help operators finish recurring work accurately, help managers understand business changes quickly, and use agents to automate analysis, reporting, task scheduling, and follow-up workflows.

## Repository Status

This repository now contains the initial product documents and the first runnable React + TypeScript console prototype.

## Development

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Run validation:

```bash
npm run build
npm run lint
```
