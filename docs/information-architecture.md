# Information Architecture

## 1. 导航原则

MAFDIC 的导航应服务高频运营动作，而不是按技术模块堆叠。

设计原则：

- 高频入口靠前
- 面向业务对象组织页面
- 查询、分析、配置、审批、AI 助手分工清楚
- 每个页面都有明确的主对象和主动作
- 支持从客户、交易、商机、风险、配置之间交叉跳转

## 2. 一级导航

### 综合工作台

页面职责：

- 汇总今日运营状态
- 展示关键经营指标
- 展示待办、风险、异常和报表任务
- 提供 AI 助手快捷入口

核心对象：

- 指标
- 待办
- 风险事件
- 报表任务
- AI 建议

### 客户全景图

页面职责：

- 展示机构客户 360 视图
- 聚合客户相关账户、交易、持仓、收益、商机和风险
- 作为客户维度下钻入口

核心对象：

- 客户
- 联系人
- 交易账户
- 基金账户
- 持仓
- 交易
- 商机
- 风险事件

### 交易与资产中心

页面职责：

- 查询交易记录
- 查看订单状态轨迹
- 查看持仓、资产、收益、分红和费用
- 解释资产变化

核心对象：

- 交易申请
- 交易订单
- 持仓
- 份额
- 资产
- 收益
- 分红
- 费用
- 基金行情

建议二级页：

- 交易记录
- 持仓资产
- 基金行情
- 收益费用
- 资产归因

### 确认与清算中心

页面职责：

- 追踪交易申请到确认结果的链路
- 查看 TA 回执、确认状态、清算记录和对账差异
- 解释未确认和差异原因

核心对象：

- 交易申请
- TA 回执
- 确认单
- 清算记录
- 对账差异

建议二级页：

- 确认状态
- 回执明细
- 清算记录
- 对账差异

### 风险异常看板

页面职责：

- 展示风险事件和告警
- 提供原因解释和处置建议
- 支持闭环跟踪

核心对象：

- 风控规则
- 风险事件
- 告警
- 处置记录

建议二级页：

- 风险总览
- 告警列表
- 处置跟踪
- 风控规则

### 经营分析看板

页面职责：

- 面向管理者展示经营指标
- 分析资产、交易、收入、费用、客户贡献、产品贡献和渠道贡献
- 支持报表生成

核心对象：

- 指标
- 客户贡献
- 产品贡献
- 渠道贡献
- 收入
- 费用
- 商机转化

建议二级页：

- 经营总览
- 收入费用
- 客户贡献
- 产品渠道
- 报表中心

### 运营配置中心

页面职责：

- 维护和查看运营规则
- 管理垫资、垫资行、孳息和费用配置
- 展示配置版本、审批状态和审计记录

核心对象：

- 垫资配置
- 垫资行
- 孳息规则
- 费用规则
- 配置版本
- 审批记录

建议二级页：

- 投资者垫资配置
- 垫资行维护
- 孳息配置
- 费用规则
- 配置审计

### 商机与业绩归因中心

页面职责：

- 管理商机生命周期
- 展示商机与客户、交易、收入和业绩的关系
- 输出业绩归因结果

核心对象：

- 商机
- 关联客户
- 关联交易
- 收入
- 费用
- 业绩归因

建议二级页：

- 商机列表
- 商机详情
- 关联交易
- 业绩归因

### 作业与案件中心

页面职责：

- 管理待办、工单和案件
- 展示处理流转和协同记录
- 作为 AI 自动建单和补全上下文的承载页

核心对象：

- 任务
- 工单
- 案件
- 处理意见
- 协同记录

建议二级页：

- 我的待办
- 工单列表
- 案件列表
- 协同记录

### 审批与授权中心

页面职责：

- 管理配置、报表、导出、AI 动作等审批
- 展示审批状态和授权边界

核心对象：

- 审批实例
- 审批节点
- 审批意见
- 授权记录

建议二级页：

- 待我审批
- 我发起的
- 审批模板
- 授权记录

### 知识与质检中心

页面职责：

- 管理 SOP、制度、FAQ、案例和质检结果
- 为 AI 提供检索增强上下文

核心对象：

- SOP
- FAQ
- 案例
- 制度文档
- 质检结果
- 整改记录

建议二级页：

- 知识库
- SOP
- 案例库
- 质检结果

### AI 助手工作台

页面职责：

- 提供对话式业务分析和动作编排入口
- 展示上下文、工具调用、动作预览和人审确认
- 支持报表、任务、邮件、配置说明、异常解释等 mock 流程

核心对象：

- 对话
- 业务上下文
- Agent 任务
- 动作预览
- 人审确认

建议二级页：

- 对话工作台
- Agent 任务
- 报表草稿
- 定时任务

## 3. 首版导航建议

第一版前端为了聚焦，可以先实现 8 个主入口：

- 综合工作台
- 客户全景图
- 交易与资产
- 风险异常
- 经营分析
- 运营配置
- 商机归因
- AI 助手

确认与清算、作业案件、审批授权、知识质检可以先作为页面内模块或占位入口，第二阶段再拆成独立一级导航。
